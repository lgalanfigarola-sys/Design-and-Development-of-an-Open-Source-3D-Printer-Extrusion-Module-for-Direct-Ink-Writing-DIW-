# Design-and-Development-of-an-Open-Source-3D-Printer-Extrusion-Module-for-Direct-Ink-Writing-DIW-
Repository featuring all necessary files to replicate Design and Development of an Open-Source 3D Printer Extrusion Module for Direct Ink Writing's findings.
- Folder "CAD" includes SLDPRT of all components and an assembly to guide the reader.
- Folder "STL" includes STL files of all components.
- File "TotalPrint" is a sliced 3mf to use as a template for printing.
- Folder "G-codes" includes all G-codes used in txt format to replicate the project's findings.
