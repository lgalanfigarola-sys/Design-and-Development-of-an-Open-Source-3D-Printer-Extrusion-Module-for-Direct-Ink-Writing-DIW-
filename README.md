# Design-and-Development-of-an-Open-Source-3D-Printer-Extrusion-Module-for-Direct-Ink-Writing-DIW-
Repository featuring all necessary files to replicate Design and Development of an Open-Source 3D Printer Extrusion Module for Direct Ink Writing's findings.
- "CAD files" includes SLDPRT of all printing components, representations of Off-The-Shelf parts and an assembly to guide the reader. OTS COMPONENTS SHOULD NOT BE PRINTED! Printed parts are: Body, ExtruderHandwheel, LateralSensorTouch, SensorHolder, PrinterMount, Syringe Holder, Syringe Holder- 5ml, Syringe Holder - 3 ml, PlungerHolder, Bearing Support, PlungerCover, BodyCover.

- "STL files" includes STL files of all printing components.
- "TotalPrint" is a sliced 3mf to use as a template for printing with a 10 ml syringe.
- "G-codes" includes all G-codes used in txt format to replicate the project's findings.
